# Examples

A collection of AkandeChips code examples.

## Beginner Example
```akandechips
# Print numbers 0 to 4
for i in range(5):
    print(i)
```

## Intermediate Example
```akandechips
# Function to check even/odd
def is_even(n):
    return n % 2 == 0

print(is_even(10))
```

## Advanced Example
```akandechips
# Simple AND gate module
module and_gate(input a, input b, output y):
    y = a & b
```
