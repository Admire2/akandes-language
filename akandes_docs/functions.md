# Functions & Built-ins

AkandeChips supports user-defined and built-in functions.

## Defining Functions
```akandechips
def add(x, y):
    return x + y
```

## Calling Functions
```akandechips
result = add(2, 3)
print(result)
```

## Built-in Functions
- `print(value)`
- `len(list)`
- `range(start, stop, step)`
- `type(obj)`

## Lambda Functions
```akandechips
square = lambda x: x * x
print(square(4))
```
