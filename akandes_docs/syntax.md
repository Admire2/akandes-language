# Language Syntax

AkandeChips syntax is designed to be familiar to Python users, with additional constructs for hardware design.

## Variables
```akandechips
a = 10
b = 3.14
c = "hello"
```

## Data Types
- Integer: `a = 5`
- Float: `b = 2.5`
- String: `c = "text"`
- Boolean: `flag = True`
- List: `nums = [1, 2, 3]`

## Operators
- Arithmetic: `+`, `-`, `*`, `/`, `%`
- Logical: `and`, `or`, `not`
- Bitwise: `&`, `|`, `^`, `~`

## Control Flow
```akandechips
if a > 5:
    print("a is big")
else:
    print("a is small")

for i in range(5):
    print(i)
```

## Comments
```akandechips
# This is a comment
```
